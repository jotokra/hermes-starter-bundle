# Hermes Agent — persona

This file is loaded into every session as the agent's persona. By default
it's empty; populate it as you develop a voice for your agent.

Examples of what to put here:
- A short bio (who the agent is, in your voice)
- Tone preferences (terse vs. verbose, formal vs. casual)
- Catchphrases or signatures
- Pronouns / how the agent should refer to itself

Keep it short — `SOUL.md` is in the system prompt, so every line costs
tokens on every turn.
